指令备忘（填坑ing）

Windows

指令 备忘 ~
python -m https.server [port] -d [指定地址] Python自带的简易http服务，文件共享用 ✔
netsh winsock reset 重置Winsock目录（需要Windows管理员权限） ✔
ipconfig/flushdns 清除DNS缓存（需要Windows管理员权限） ✔

Linux

指令 备忘 ~
scp [-1246BCpqrv] [-c cipher] [-F ssh_config] [-i identity_file] [-l limit] [-o ssh_option] [-P port] [-S program] user1@address:file1 user2@address:file2 用于 Linux 之间复制文件和目录 ✔
pwd 查看当前路径 ✔
